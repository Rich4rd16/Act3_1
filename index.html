HTML

Es el estándar utilizado para crear y diseñar la estructura de una página web. Proporciona una estructura jerárquica de elementos mediante etiquetas para definir y organizar contenido como encabezados, párrafos, imágenes, enlaces y más.

En 02

CATEGORY

CSS

Es un lenguaje de estilo utilizado para controlar la presentación y el diseño de documentos HTML. Permite a los desarrolladores definir la apariencia visual de los elementos, especificando atributos como colores, fuentes, espaciado y posicionamiento para crear diseños web visualmente atractivos y receptivos.

Ricardo Lazo
En 03

CATEGORY

JAVASCRIPT

Es un lenguaje de programación versátil utilizado principalmente para scripts del lado del cliente en desarrollo web. Permite actualizaciones de contenido dinámicas, interacción con usuarios y manipulación del Modelo de Objetos del Documento (DOM). JavaScript es esencial para crear páginas web interactivas y receptivas, mejorando la experiencia y funcionalidad del usuario.

Carlos Ipiales

REACT

Es una biblioteca de JavaScript desarrollada por Facebook para construir interfaces de usuario interactivas y reutilizables. Se centra en la creación de componentes que gestionan su propio estado y se combinan para formar interfaces complejas. React utiliza un modelo de programación declarativa y virtual DOM para mejorar el rendimiento de las actualizaciones de la interfaz de usuario.

ANGULAR

Es un marco de desarrollo web de código abierto desarrollado y mantenido por Google. Utiliza TypeScript y sigue el patrón de diseño de arquitectura de componentes. Angular proporciona un conjunto completo de herramientas para construir aplicaciones web robustas y escalables, incluyendo enlaces bidireccionales, inyección de dependencias y un potente sistema de módulos.

VUE

Es un marco progresivo de JavaScript para construir interfaces de usuario. Es liviano y fácil de integrar con otros proyectos. Vue se destaca por su enfoque incremental, lo que significa que puede ser adoptado gradualmente en proyectos existentes. Ofrece una sintaxis clara y concisa, así como una excelente documentación, facilitando el desarrollo de aplicaciones web interactivas y eficientes.
